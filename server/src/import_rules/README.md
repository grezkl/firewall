A shell script which can import firewall rules(text) to database.

## usage:
```
$ chmod +x import_rules.sh

$ ./import_rules.sh <the file of rules>
```

## notes
The format of rules should be like this:
```
<program_name1>
<program_path1>
<program_name2>
<program_path2>
<program_name3>
<program_path3>
...
```
